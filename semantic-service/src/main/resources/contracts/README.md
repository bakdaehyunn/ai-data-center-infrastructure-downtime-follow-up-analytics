# Contract Resource Placeholder

This directory reserves the future service contract resource boundary.

Canonical Phase 8-9 contract artifacts remain at:

- `semantic-service/boundary-contract.ttl`
- `semantic-service/openapi.semantic-service.yaml`
- `semantic-service/api-dtos.md`

Phase 10 adds `semantic-service-contracts.ttl` as a parseable manifest that
points back to those contracts. It does not copy contracts into packaged
resources or run code generation.
